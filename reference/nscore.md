# Gaussian Transformation (Normal Score)

Originally written by Ashton Shortridge, May/June, 2008.

## Usage

``` r
nscore(x)
```

## Arguments

- x:

  Numeric vector to transform.

## Value

A list containing:

- nscore:

  The normalized score vector.

- trn.table:

  A data frame mapping the sorted original `x` values to the sorted
  `nscore` values.
